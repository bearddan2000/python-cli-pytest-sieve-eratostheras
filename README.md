# python-cli-pytest-sieve-eratostheras

## Description
Print all primes smaller than or equal to n
POC for pytest.

## Tech stack
- python
  - pytest

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
